Allow users to authenticate with their nicknames as well as full usernames. THIS IS ALMOST CERTAINLY A SECURITY NIGHTMARE. Consider carefully before enabling.

== Description ==

After activating this plugin, users can authenticate using either their username or their nickname. Nicknames that are set after activation are 
forced to be unique.

Known Bugs: 

  - The plugin will not check for duplicate nicknames on installation. Please ensure that nicknames are unique, otherwise there logins will be messed up.


== Installation ==

Installing "nickname-login" can be done by downloading the .zip file and uploading it through 
the 'Plugins > Add New > Upload' screen in your WordPress dashboard. Then activate the plugin through the 'Plugins' menu in WordPress.
